# 🚀 Web Fundamentals - Hora do TI

Bem-vindo ao repositório central da disciplina de Web Fundamentals. Este espaço organiza a jornada de aprendizado em desenvolvimento web, versionamento e gestão de projetos.

## 📂 Estrutura do Repositório

O projeto está dividido em três pilares principais:

* **[📁 Apresentação](./apresentacao):** Landing page inicial contendo o perfil do instrutor e objetivos da disciplina.
* **[📁 Exercícios](./exercicios):** Laboratórios e desafios práticos desenvolvidos em sala de aula.
* **[📁 Registros](./registros):** Portfólio dinâmico construído em **React + Vite**, demonstrando o domínio de ferramentas modernas de Front-end.

## 🛠️ Tecnologias Utilizadas

* **Node.js** (Ambiente de execução)
* **React** (Biblioteca para interfaces)
* **Vite** (Ferramenta de build rápida)
* **Git & GitHub** (Versionamento)
